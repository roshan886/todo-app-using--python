📝 Python Todo App

A simple command-line based Todo Application built using Python.
This project allows users to manage daily tasks efficiently with basic CRUD operations.

🚀 Features

- ➕ Add new tasks
- 📋 View all tasks
- ✏️ Update existing tasks
- ❌ Delete tasks
- 🧠 Simple and beginner-friendly logic

🛠️ Technologies Used

- Python 3

📂 Project Structure

todo-app/
│
├── main.py
├── README.md

▶️ How to Run

1. Clone the repository

git clone https://github.com/roshan886/todo-app-using--python.git

2. Open the project folder

cd todo-app

3. Run the Python file

python main.py

💡 Example Menu

1. Add Task
2. View Tasks
3. Update Task
4. Delete Task
5. Exit
